## Elasticsearch Wiki

## 概述

Elasticsearch是一款使用Java语言基于Apache Lucene开发的分布式全文搜索引擎，从版本8开始支持向量检索，主要应用于搜索推荐、日志分析、向量检索、大数据hadoop集成等场景。在DB-Engines上排名搜索引擎第二，常年排名第十。本作者结合多年奋斗在互联网一线的实战经验，结合在实际工作中遇到的各种问题，整理成文档，希望为开源社区，还有新入职场的DB小白贡献自己的一点点力量。
